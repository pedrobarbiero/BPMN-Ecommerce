# BPMN-Ecommerce

Mapeamento de processo em BPMN de aquisição de produtos de um e-commerce, considerando os produtos já selecionados até o despacho pela transportadora. Sobre a notação deve conter pelo menos os seguintes itens:


2 Piscinas;
Raias representando as áreas dentro da organização;
4 tipos de desvio de fluxo;
atividades ou subgrupos repetitivos;
Eventos de timer;
Indicar em algum comentário a empresa que foi analisada como modelo.
